About our Project

This Website is an assignment for Information Systems 601 taught by Professor Keith Williams. This website contains the multiple tabs each explaining different things. It starts from home page all the way to the gist of the internet.

The Home page provides an Outline of events which lists summaries for the 5 videos included in this website/project. Clicking on the summary of a particular video leads us to the summary of that particular video. It also has links to the gist of the Internet and also to view the collaborators involved.

The next tab shows the team involved in building this project along with the links to their GitHub accounts. Clicking on the GitHub icon below each of the team members involved will lead their GitHub accounts respectively.

The next is about us page(this) which explains about this course/projects, professor and outline of this project. It also mentions the tools and technologies used and the learning from this project.

The next tab is Videos. The dropdown lists the 5 different videos which were provided to us and clicking on them leads us the summaries of that particular video. The first video talks about "The invention of the internet", the second one provides a summary for "Browser Wars" which also talks about the Mosaic Browser, Navigator browser, negotiations between the Microsoft and also the launch of Internet Explorer. The third video summary talks about "Internet Search" which is about the launch of yahoo and Google and also the people involved. The fourth video summary "DotCom Bubble" talks about E-Commerce, Amazon, eBay and how Cryptography played a crucial role in E-Commerce. The last Video summary "People Power" talks about the people, the world wide web brought forth, YouTube and mp3. The last menu from the dropdown outlines the gist of the History of Internet.

All in all, we are here demonstrating what we understood about the basics of how the internet works/functions and its roots, with the use of Bootstrap, Webstorm and Git. It is to hone out website creation skills.
